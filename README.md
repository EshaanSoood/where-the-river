## Dream River Web

Quickstart:

1. Copy `.env.example` to `.env.local` and set Supabase + Plausible vars.
2. Run `npm run dev` in `web/`.
3. Visit `/participate` for magic link signup.

Deploy: Vercel recommended. Set env vars in project settings.
